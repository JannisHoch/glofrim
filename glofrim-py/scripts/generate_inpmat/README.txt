to compile generate_inpmat follow do these steps:
-Copy the 'sample_Makefile', 'run-makefile.sh' and 'generate_inpmat.F' to the map folder of your model
-Make sure the map folder should contain a params.txt and a hires folder with a location.txt file and a *.catmxy for model each region.
-Adapt the sample_Mkinclude include file to work for your environment
-Compile the script by running run-makefile.sh <path/to/MKinclude>
